# echo-word

Inspired by `fortune`, prints a random verse from the Bible to the terminal.

## usage

1. Get the plain ASCII text KJV version from [ebible.org](http://ebible.org/kjv/)

2. Unzip the files somewhere

3. `echo-word <path-to-files>`
